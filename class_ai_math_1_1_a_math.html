static unsigned int AiMath::AMath::GetNextPowerOfTwo ( const unsigned int  a_value)
inlinestatic

Get next power of two if not already power of two. Else return given value.

copy the most significant bit to all lower bits, then add 1 which results in carry that sets all lower bits to 0 and one bit beyond to 1. If the original number was a power of 2 then the decrement will reduce it by one less, so then round up to original value.
